# Excel-Diary-Maker
Program in Python which make excel diary from your notes and update them with new ones by adding them at the end.<br>
Notes have format: <br>
  - "Day" Date Title<br>
  - (body)<br>
  - "Day" Date Title<br>
  - (body)<br>
  
Program use openpyxl from Python
