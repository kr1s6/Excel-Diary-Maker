# Excel-Diary-Maker
<h3>To use app just download file Excel-Diary_Maker.exe</h3>
<h4>This is not a professional program, so be sure to make a copy before updating your journal! Just in case if something unexpected would destroy diary.</h4>
Program in Python which make excel diary from your notes<br>
and update them with new ones by adding them at the end.<br>

Notes have format:
  - "Day" Date Title<br>
  - (body)<br>
  - "Day" Date Title<br>
  - (body)<br>
  
Program use openpyxl from Python
